**Context:** Beginning this project, I did not use any AI to code at this stage. I was interested in understanding the project's core objectives and how they relate to the tasks we completed over the past 8 weeks. 

The goal of the project is to build a smart finance assistant. These are the main objectives that the assistant must accomplish:

- Summarising transactions from a CSV file
- Tracking a stock/crypto portfolio via API data
- Analysing budgets and spending patterns
- Offering advice through a chatbot personality
- Running a simple custom tool (e.g., currency converter, savings goal calculator)

**My Prompt:** The AI I used to help broaden my understanding of the assessment was Gemini.

"Hello Gemini, I am a first-year, second-semester Business
Information Systems undergraduate. This semester, I am taking a module called
Introduction to Business Programming, and we have been learning to code in
Python. Over the past few weeks, we have been learning how to create a finance
tracker, and my second assessment is to build a smart finance assistant using
Python, Google Colab, Gradio, and the hands-on-ai package. For this
assignment, I have to collaborate with an AI in creating my finance assistant.
These are the main objectives of the assistant:
- Summarising transactions from a CSV file
- Tracking a stock/crypto portfolio via API data
- Analysing budgets and spending patterns
- Offering advice through a chatbot personality
- Running a simple custom tool (e.g., currency converter,
savings goal calculator)
I would like you to help break down what you think the assessment
is expecting of me how to best achieve it."

**AI Response Summary:** The AI broke down each stage of the objectives and how to possibly achieve them. At my request, it also gave me a list of all the sources used in its response.
